# FATE-Vault
Its a passion project to collect and share FATE TTRPG ideas of characters, stunts

Made by Oleh Ruban "irik1"

Technologies used:
Backend - Golang with Gin framework
Frontend - Vue.js with assistanse of Cursor Ai agent
Database - MongoDB


Stages of development
1. Data stucture and basic backend routes for getting list of charsheets
2. Frontend logic and website layout of charsheets list
3. Charsheets creating and editing logic for full characters, NPCs and Mooks. Create a few basic charsheet presets for Fate Core, Fate Accelerated and Fate Condensed
4. User logic - registration/login, personal charsheets and sharing logic
5. More distinctive site layout for different type of charsheets and set space for stunt database
6. Stunt backend logic to allow create, edit and share character stunts
7. Add possibility to add to charsheet stunt from database instead of manual creating
8. Add more customization of charsheets to allow add more Extra stuff to charsheet
9. Add printing of charsheets with basic presets in PDF with filled fields from data
10. Add docker to create container of backend service
11. Try to include kubernetes to create backend servise pods
12. If finished project is succesfull - host web app on domain. In case if result isn't interesting or usable - host web app with port-forward technique would be enough
